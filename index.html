Primeiros Computadores

A primeira aula tem o intuito de introduzir a computação de maneira geral, apresetando a história da web com os primeiros computadores, como surgiram e grandes nomes do ramo.

A começar pelos primeiros computadores a introdução é realizda com a primeira geração, onde temos o Harvard Mark I (1944), Colossus(1946) e o ENIAC (1946) que podem se resumir basicamente em grandes maquinas para calculos complexos, muito utilizados durante a 2° guerra para criptografar e descriptografar menssagens. Inclusive foi durante esta época que o termo "bug" (que era uma maneira de dizer que o computador estava com uma série de erros, por haver muitos insetos atraidos pelo calor que as maquinas geravam enquanto ligadas), foi popularizado na computação.

Logo após a primeira geração, tivemos a segunda e terceira geração de computadores. A Segunda geração ficou marcada pela criação dos transistores, o que possibilitou a diminuição dos computadores, se antes eles tinham o tamanho de uma sala inteira agora eles tinham o tamanho apenas de uma mesa. Já na terceira geração tivemos a criação do circuito integrado com Transistor, capacitores e resistores, o que diminuiu ainda mais o tamanho dos computadores.

Temos também a quarta geração de computadores (atual geração), com a criação dos PC's (ou Personal Computers) os computadores ficaram mais populares e mais acessiveis para as pessoas, a maior inovação dessa geração fica por conta do Mouse, notebooks e os smartphones.

E por ultimo temos a 5º geração, é uma era da tecnologia que ainda não chegou mas já podemos ter uma noção do que provavelmente vamos encontrar. As tecnologias: inteligência Artificial, computador quântico, blockchain e metaverso são as que nos dão um veslubre do futuro que nos aguarda.

Como surgiu a Internet?

A internet surge a partir de uma ideia que tornaria possivel a comunicação entre computadores que estivessem lugares completamente diferentes. A criação da primeira rede de comunicação entre computadores, foi feita apenas para instituições militares e instituições de pesequisa dos Estados Unidos. Após a criação do protocolo TCP/IP e de sua unificação com a WWW (O primeiro navegador), a internet se tornou aos poucos o que conhecemos hoje.

Client x Server

Client.

O Cliente pode ser resumido como o lado que consome a informação que está hospedada no servidor, essas informações são baixadas no computador através de um navegador (Chrome, Safari, Firefox, Edge...) que se comunica com o servidor que está hospedando o site, o navegador por sua vez baixa essas informações e as apresenta para o cliente em arquivos com formato HTML, CSS e Javascript. Considerando que cada cliente está acessando o site por algum dispositivo, (Notebook, tablet, smartphone...) este por sua vez se comporta de maneira diferente em cada um deles, o comportamento em questão é denominado responsividade.

Aplicações WEB.

As aplicações WEB podem ser descritas como aplicações que são utilizadas por um navegador, ou seja, sites como Youtube, Facebook, Twitter, Linkedin, Instagram e etc, são considerados aplicativos WEB.

Server.

O Servidor pode ser resumido como o lado que armazena alguma informação tanto numa rede local (LAN) quanto na internet para logo após ser consumida pelo cliente. Os servidores são utilizados para hospedar sites, armazenar arquivos e aplicações WEB, e fazer também o "streaming" das informações contidas nele.

Os servidores diferentemente das maquinas para o usuário final, possuem sistemas operacionais especificos para utilização e servidores web para hospedagem de aplicações WEB.

Tipos de servidores

  • Application Server
  • Web Server
  • Database Server
  • Firewall
  • Proxy
  • DNS

Linguagem de Programação

Para a Construção de um software é necessário escrever um conjunto de instruções e regras determinando uma sequência lógica para a execução, para tal tarefa utilizamos uma ferramenta chamada Linguagem de Programação.

As Linguagens de programação são utilizadas para desenvolver softawares, para isso existem diversos tipos de linguagens de programação que nos possibilitam escolher a maneira de construir o software são alguns exemplos: Javascript, Python, Java, PHP, Dart, GO, Swift e entre muitas outras...

Cada uma das linguagens citas acima possui uma utiizção especifica, seja no lado do Servidor ou no lado do cliente. As linguagens utilizadas no lado do servidor irão fazer toda a lógica e todo o processamento, como: Java, C#, PHP. Já as linguagens no lado do cliente a mais conhecida é o Javascript (Que pode ser utilizado tanto no lado do servidor quanto no cliente através do V8 Engine), o que é apresentado ao usuário é apenas HTML, CSS e Javascript.

Introdução ao HTML na Prática

(Voltar)

Começamos o curso de HTML realizando os primeiros passos básicos: instalando o nosso editor de texto. É bom frisar que o HTML não é uma linguagem de programação e sim uma linguagem de marcação de Hipertexto voltando a instalação, podemos utilizar qualquer editor de texto para escrevermos nosso código HTML, como: Notepad++, Sublime text, Visual Studio Code, entre outros... Para realização do curso utilizaremos o Visual Studio Code, pela facilidade e praticidade do programa para instalar os "plug-ins" necessários para realizarmos as atividades.

No curso aprendemos a inspecionar os elementos das páginas, ou seja, conseguimos ir em qualquer página na web e vizualizar a esturutra HTML e CSS construida por trás da página, também é possivél fazer edições nos elementos das paginas, possibilitando assim a pratica e nos dando um maior entendimento sobre como funciona o HTML.

Inciando as aulas de HTML começamos a ver as tags principais para a construção da estrutura básica, dentre elas estão presentes: <"html">; <"head"> e <"body">. Essas tags compõem a estrutura básica e dão formato a pagina, a tag "html" refere-se a todo o código que será inserido no documento, a tag "head" é onde pode ser inserido elementos que o navegdor irá carregar antes de exibir o conteudo da pagina, e a tag "body" é todo o corpo do documento, ou seja, é tudo o que o usuário pode ver e interagir. Ainda sobre as Tags, temos outras como: <"p"> para paragrafos; <"i"> para deixarmos o texto em itálico; <"h1"> para titúlos; <"a"> para "ancorar" um link a um texto, entre outras... E se tratando de tags temos também os atributos, eles servem para adicionar uma função especifica a uma determinda tag, e com isso podemos atribuir estilos, identificadores, classes, entre muitos outros.
